### How To Use
---
> **Warning** : 이 레포지토리를 클론, 실행하기 위해서는 `git`, `Node.js` 그리고 `yarn`이 설치되어 있어야 합니다.

```
# clone this repository
$ git clone https://github.com/jeonhaekang/portfolio-page.git

# into the repository
$ cd portfolio-page

# install dependencies
$ yarn install

# run development mode
$ yarn dev

# build
$ yarn build
```
